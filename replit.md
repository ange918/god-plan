# God Plan

## Overview
A Next.js 16 application using React 19, TypeScript, and Tailwind CSS 4. This is a fresh starter project.

## Project Architecture
- **Framework**: Next.js 16 (App Router)
- **UI**: React 19, Tailwind CSS 4
- **Language**: TypeScript 5
- **Build Tool**: Turbopack (Next.js built-in)

### Directory Structure
- `app/` - Next.js App Router pages and layouts
- `public/` - Static assets

## Development
- Dev server runs on port 5000 via `npx next dev --hostname 0.0.0.0 --port 5000`
- `next.config.ts` is configured with `allowedDevOrigins` for Replit proxy compatibility
